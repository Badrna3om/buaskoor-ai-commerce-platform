# Setup Guide

Requirements:
- n8n
- OpenAI API
- Evolution API or another WhatsApp provider
- WordPress with WooCommerce
- Telegram bot
- n8n Data Table named `chat_sessions`

After importing:
1. Configure all credentials.
2. Replace Data Table placeholders.
3. Replace product and variation IDs.
4. Configure WooCommerce REST access.
5. Set Telegram chat ID.
6. Link the main workflow to the order-creation sub-workflow.
7. Review shipping rules.
8. Test before activation.
